# Caesar Cipher Web App

A simple web application that allows you to encrypt and decrypt messages using the classic **Caesar Cipher** technique. Non-alphabet characters remain unchanged, and you can customise the shift value.

---

## Features

- Encrypt messages with a custom shift.
- Decrypt messages with the same shift.
- Clear input and output fields.
- Handles both uppercase and lowercase letters.
- Responsive and visually appealing design using Tailwind CSS.
- Shift works with negative numbers too.
